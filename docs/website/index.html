Trylid

What is Trylid?

Trylid is a computer language. It is basically a cross between Python and Smalltalk, running on Fonc.

Getting Trylid

Trylid is available as a tarball. The Git source repository is available at repo.or.cz, but you'll probably want to start with the tarball (unless you just want to browse the source).

Documentation

The documentation is, to put it nicely, "concise". To put it another way, "terse and no doubt incomplete".
